# Мапа тривог (клон nahr.uk) — покрокове розгортання

## 1. Створи приватний GitHub-репозиторій
Завантаж туди всі файли з цієї папки (структуру не міняй).

## 2. Отримай Telegram API-ключі
1. Зайди на https://my.telegram.org → API development tools
2. Створи застосунок, збережи `api_id` та `api_hash`

## 3. Отримай string session (один раз, локально на своєму комп'ютері)
```bash
pip install telethon
python generate_session.py
```
Введи api_id, api_hash, потім номер телефону і код підтвердження з Telegram.
Скрипт видасть довгий рядок — це і є `TG_SESSION`.

⚠️ Цей рядок = повний доступ до твого акаунта. Нікому не показуй, зберігай тільки в GitHub Secrets.

## 4. Додай секрети в репозиторій
Settings → Secrets and variables → Actions → New repository secret:
- `TG_API_ID`
- `TG_API_HASH`
- `TG_SESSION`

## 5. Увімкни GitHub Pages
Settings → Pages → Source: `main` branch, папка `/docs`.
Через хвилину карта буде доступна за адресою на кшталт:
`https://<твій_нік>.github.io/<репозиторій>/`

## 6. Увімкни Actions
Вкладка Actions → якщо попросить підтвердити workflow — підтверди.
Скрипт `scrape.yml` буде запускатись кожні 5 хвилин, читати канали,
оновлювати `events.json` і комітити зміни. Карта на GitHub Pages
підхоплює ці дані автоматично (оновлюється сама раз на хвилину).

Можна запустити вручну: Actions → Scrape Telegram alerts → Run workflow.

## Важливо розуміти про якість розпізнавання

`scraper.py` зараз має **приблизний** парсер:
- визначає тип загрози (ракети/БПЛА/авіація/обстріл/тривога/відбій) за ключовими словами
- намагається витягти назву населеного пункту за прийменниками ("в Ростові", "над Бєлгородом")
- геокодує назву через безкоштовний Nominatim (OpenStreetMap)

Реальний формат повідомлень у кожному з 5 каналів може відрізнятись
(емодзі-мітки, скорочення областей, специфічні шаблони). Щоб точність
була високою — треба:
1. Дати мені кілька реальних прикладів повідомлень з кожного каналу
2. Я підправлю регулярки в `TYPE_KEYWORDS` і `LOCATION_PATTERNS` під конкретний формат

Без цього карта запрацює, але частина подій може лишитись без
координат (location_name: null) або з неправильно визначеним типом.

## Обмеження безкоштовних тарифів
- GitHub Actions: 2000 безкоштовних хвилин/міс на приватних репо
  (для публічного репо — необмежено). Запуск кожні 5 хв ≈ 8600 запусків/міс,
  кожен займає секунди-десятки секунд — має вкладатись з запасом.
- Nominatim: ліміт ~1 запит/секунду, скрипт це враховує (`time.sleep`).
  Кеш геокодування (`geocode_cache.json`) означає, що кожне місце
  геокодується лише один раз.
- GitHub Pages: повністю безкоштовний для публічних репо.
